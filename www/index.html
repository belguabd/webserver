What is Webserv?

Discover the architecture behind our lightning-fast HTTP server

Built for Performance

Webserv is a high-performance HTTP server crafted entirely in C++98, designed with modern web standards in mind. Our server handles thousands of concurrent connections using advanced non-blocking I/O techniques.

server {
listen 8080;
server_name example.com;
root /var/www/html;
location / {
allowed_methods GET POST;
}
}

Powerful Features

Our server is packed with features to handle any modern web application's needs.

  • Non-blocking I/O with kqueue/select/ekqueue
  • Multi-port listening and virtual hosts
  • Full CGI execution (PHP, Python, etc.)

Robust Architecture

Get Started in Minutes

Deploy your own high-performance HTTP server with just a few commands

1

Clone & Build

$ git clone repository
$ make all
2

Configure

server {
listen 8080;
root /var/www;
}
3

Launch

$ ./webserv config.conf
🚀 Server running on port 8080!
